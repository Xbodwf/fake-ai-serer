# 管理员设置

## 默认管理员账户

系统首次启动时，会自动创建一个默认管理员账户：

- **用户名**: `admin`
- **密码**: `admin123`
- **初始余额**: $10,000

## 如何使用

1. 启动服务器：
   ```bash
   pnpm dev
   ```

2. 打开浏览器访问 `http://localhost:5173`

3. 点击"登录"，使用以下凭证登录：
   - 用户名: `admin`
   - 密码: `admin123`

4. 登录后，点击导航栏中的"Admin Console"进入管理员控制台

## 管理员功能

在管理员控制台中，你可以：

- **查看系统统计**: 用户数、API Key 数、模型数等
- **查看使用分析**: 总余额、平均余额、总使用量、总成本
- **管理用户**: 查看所有用户、编辑用户信息、充值/扣费

## 安全建议

- 首次登录后，建议修改默认密码
- 不要在生产环境中使用默认密码
- 定期检查用户活动和使用情况

## 创建新用户

普通用户可以通过注册页面自行创建账户。注册后的用户默认为普通用户角色。

如需将用户升级为管理员，请直接编辑 `data/users.json` 文件，将用户的 `role` 字段改为 `"admin"`。
